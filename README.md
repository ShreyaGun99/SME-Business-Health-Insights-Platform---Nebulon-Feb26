# SME Business Health Insights Platform

This project focuses on analyzing SME business data to generate actionable insights and support data-driven decision-making. 

It includes data preprocessing, dashboard and website codes, feature engineering designed to evaluate business performance and predict key outcomes.

The solution aims to help identify trends, improve efficiency, and enable better strategic planning.

- Python
- Data Analysis
